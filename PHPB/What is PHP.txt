PHP is a server side programming language that is mostly used for web applications.

The term PHP is an acronym for Hypertext Preprocessor. It is an open-source, interpreted, object-oriented server-side scripting language that is used for web development. PHP is developed by the Rasmus Lerdorf in 1994 with the very first version of PHP that simply designed to set the Common Gateway Interface (CGI) binaries, which are written in C programming language.

There are many advantages of choosing PHP language over others. In this PHP Tutorial, you will learn more about PHP Usage, Feature, History, and Future scope.

EXAMPLE: Simple program to print “Hello world!” message on the screen.

        <?php

        /* echo is a print command */
        echo "Hello world!"; 

        ?> 
    
OUTPUT:

        Hello world!
